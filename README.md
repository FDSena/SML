# SML — Small Language Model

SML est un projet autour des Small Language Models, avec pour objectif d’explorer, implémenter et documenter les principes de base des modèles de langage légers.

## Objectifs

- Comprendre le fonctionnement d’un modèle de langage
- Expérimenter avec l’entraînement ou l’utilisation d’un petit modèle
- Organiser le code source et les notebooks
- Documenter les résultats et les choix techniques

## Contenu du dépôt

- Code source du projet
- Notebooks d’expérimentation
- Documentation
- Résultats et observations
